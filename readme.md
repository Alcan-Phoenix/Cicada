# Cicada 框架整体介绍
===================
首先得以肯定的是，这是一个很容易上手开发的系统框架，使开发人员不需要过多地去关注系统相对底层的一些开发，
让整个开发的注意点都落实到业务实现上。同时这样也保证了系统底层架构的统一性，可维护性，程序的可拓展性等特点。
